You can find the continuation <a href="https://github.com/towerwatchman/Atlas" >here</a> # Atlas is an open source game manager and launcher for Windows and Linux with a modern interface for viewing, finding, and easily installing 3rd party games and visual novels. We currently support adding games from <a href="https://www.dlsite.com/" >DLSite</a> and plan on adding more sources as the project progresses (See features below). It's meant to replace <a href="https://www.game-list.org" >Game-List</a> (<a href="https://github.com/game-list/game-list" >Github</a>) but also expand on it's feature set. Atlas is for the user that wants ease of use, quality of life and have the software do the heavy lifting for you. For discussion about future releases and new features please visit our <a href="https://discord.gg/XpTHvYbYyz" >Discord Server</a>. @subsection autotoc_md1 Features: - Fast and Snappy UI. - Easy to understand UI written in Qt6. - Custom themes and color schemes. - Game file management (moves or copies game install files for you) - Easy to edit metadata and tags per title. - Navigate titles with a Banner/Thumbnail gallery grid (Images are not automatically downloaded yet). - View or add previews for a title. - Filter titles based on titles, creators, tags, ID's, description contents and urls. - Bulk install/import (See <a href="https://github.com/KJNeko/Atlas/issues/6" >Issue #6</a>] multiple titles and versions with a single approval. - Compresses all banner and preview images to smaller webp files for storage optimization. <img src="https://github.com/KJNeko/Atlas/assets/36950324/4063b991-4e36-43dc-b695-6bf85b782770" alt="image"/> @subsubsection autotoc_md2 DLSite integration <a href="https://github.com/KJNeko/Atlas/issues/3" >Issue #3</a> - [ ] Scrape Metadata (titles, tags, description, version and image urls) - [ ] Game updates/notifications - [ ] Follow your popular creators for new releases @subsubsection autotoc_md3 Steam/SteamCMD integration <a href="https://github.com/KJNeko/Atlas/issues/4" >Issue #4
