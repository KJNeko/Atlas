Detailed Description

template<std::size_t N>
requires ( N > 0 )
struct fgl::string_literal< N >

A wrapper type for string literals.

Todo:
i should rethink this https://www.youtube.com/watch?v=ABg4_EV5L3w

Due to how C++ works, string literals cannot be passed as non-type template arguments. This wrapper type helps to work around that limitation.

Template Parameters
Nthe number of characters in the string literal including null-terminator, which is used as the C array extent. It must be greater than zero. This extent should be automatically deduced by the constructor.
